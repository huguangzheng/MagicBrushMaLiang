# 神笔马良 - MagicBrush

一个功能丰富的在线画图应用,支持多种绘图工具、图层管理、AI配色优化、网格显示、缩放等功能。

## 功能特性

### 1. 基本绘图工具
- **不规则线条**: 自由绘制线条
- **橡皮擦**: 擦除绘制内容(默认5px粗细,不留痕迹)
- **直线**: 绘制直线,实时显示长度
- **矩形**: 绘制矩形,实时显示尺寸
- **圆形**: 绘制圆形,实时显示半径和圆心
- **椭圆**: 绘制椭圆,实时显示尺寸
- **五角星**: 绘制五角星,实时显示尺寸
- **选择工具**: 虚框选择、移动、删除元素

### 2. 颜色和画笔设置
- 颜色选择器: 选择任意颜色
- 画笔大小调节: 1-50像素可调
- 实时预览画笔大小
- 快速颜色选择: 10种常用颜色

### 3. 图层管理
- 添加/删除图层(最多16层)
- 图层顺序调整(上移/下移)
- 图层可见性切换
- 图层锁定功能
- 多图层独立编辑
- 图层缩略图预览
- **图层颜色设置**: 每个图层可独立设置颜色(透明/白色/灰色/黑色/蓝色/浅红/浅绿/浅蓝)
- **背景图层保护**: 初始图层不允许删除、不允许上移、名称不可变
- **新建图层默认透明**
- **颜色叠加显示**: 画布显示最顶层非透明图层的颜色

### 4. 画布控制
- **缩放功能**: 50%-300%缩放范围
- **网格显示**: 显示/隐藏网格,支持5px/10px/20px/50px大小
- **网格跟随缩放**: 网格自动跟随画布缩放
- **标尺功能**: X轴和Y轴标尺,原点在画布中心,支持缩放和滚动同步
- **图层颜色**: 统一的颜色设置入口,选择图层后设置颜色
- **中心原点标记**: 画布中心显示红色十字标记和O(0,0)坐标
- **正交虚线**: 鼠标位置显示正交虚线和坐标

### 5. 编辑功能
- 撤销操作
- 清除整个画布
- 清除当前图层
- 历史记录管理
- DEL键删除选中元素

### 6. 元素选择和操作
- **虚框选择**: 拖动鼠标绘制选择框，框内元素自动被选中；双击左键取消批量选中
- **选择框显示**: 蓝色虚线边框和半透明填充
- **元素移动**: 拖动移动选中元素
- **元素删除**: DEL键删除选中元素
- **鼠标形状**: 选择元素时显示小手形状

### 7. 存档和导出
- 自动保存: 每30秒自动保存到本地存储
- 导出工程: 保存为JSON格式
- 导入工程: 从JSON文件恢复项目
- **导出PNG**: 将画布导出为PNG格式图片
- **导出JPG**: 将画布导出为JPG格式图片

### 8. AI功能
- AI配色优化: 使用大模型智能优化当前图层的配色方案（todo）

## 快速开始

### 方法1: 直接打开
1. 双击 `index.html` 文件在浏览器中打开
2. 开始绘图

### 方法2: 使用本地服务器
```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 使用Node.js的http-server
npx http-server
```

然后在浏览器中访问 `http://localhost:8000`

## 使用说明

### 绘图工具
1. 从左侧工具栏选择绘图工具
2. 选择颜色和画笔大小
3. 在画布上拖动鼠标进行绘制

### 图层管理
1. 右侧面板显示所有图层
2. 点击图层可切换当前编辑图层
3. 使用图层控制按钮进行管理操作
4. 每个图层可独立设置颜色

### 网格和缩放
1. 使用缩放按钮或鼠标滚轮缩放画布
2. 点击网格按钮显示/隐藏网格
3. 选择网格大小(5/10/20/50px)
4. 网格自动跟随缩放调整

### 元素选择和移动
1. 选择"选择"工具
2. 点击或靠近元素自动选中
3. 拖动移动元素位置
4. 按DEL键删除选中元素

### AI配色优化
1. 选择要优化的图层
2. 点击"AI优化配色"按钮
3. 等待AI处理完成
4. 查看优化后的配色效果

### 保存和加载
- 应用会自动每5秒保存一次
- 点击"导出工程"手动保存项目文件
- 点击"导入工程"加载之前保存的项目
