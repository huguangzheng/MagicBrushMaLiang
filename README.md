# 魔法画笔 - MagicBrush

一个功能丰富的在线画图应用,支持多种绘图工具、图层管理、AI配色优化等功能。

## 功能特性

### 1. 基本绘图工具
- **画笔**: 自由绘制线条
- **橡皮擦**: 擦除绘制内容
- **直线**: 绘制直线
- **矩形**: 绘制矩形
- **圆形**: 绘制圆形

### 2. 颜色和画笔设置
- 颜色选择器: 选择任意颜色
- 画笔大小调节: 1-50像素可调
- 实时预览画笔大小

### 3. 图层管理
- 添加/删除图层
- 图层顺序调整(上移/下移)
- 图层可见性切换
- 多图层独立编辑
- 图层缩略图预览

### 4. 编辑功能
- 撤销操作
- 清除整个画布
- 清除当前图层
- 历史记录管理

### 5. 存档和导出
- 自动保存: 每5秒自动保存到本地存储
- 导出工程: 保存为JSON格式
- 导入工程: 从JSON文件恢复项目

### 6. AI功能
- AI配色优化: 使用大模型智能优化当前图层的配色方案

## 快速开始

### 方法1: 直接打开
1. 双击 `index.html` 文件在浏览器中打开
2. 开始绘图

### 方法2: 使用本地服务器
```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 使用Node.js的http-server
npx http-server
```

然后在浏览器中访问 `http://localhost:8000`

## 使用说明

### 绘图工具
1. 从左侧工具栏选择绘图工具
2. 选择颜色和画笔大小
3. 在画布上拖动鼠标进行绘制

### 图层管理
1. 右侧面板显示所有图层
2. 点击图层可切换当前编辑图层
3. 使用图层控制按钮进行管理操作

### AI配色优化
1. 选择要优化的图层
2. 点击"AI优化配色"按钮
3. 等待AI处理完成
4. 查看优化后的配色效果

### 保存和加载
- 应用会自动每5秒保存一次
- 点击"导出工程"手动保存项目文件
- 点击"导入工程"加载之前保存的项目
