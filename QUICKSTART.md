# 魔法画笔 - 快速启动指南

## 🚀 30秒快速开始

### Windows用户
1. 双击 `start.bat` 文件
2. 等待服务器启动
3. 浏览器自动打开 `http://localhost:8000`
4. 开始绘画!

### Mac/Linux用户
1. 打开终端
2. 进入项目目录: `cd MagicBrushMaLiang`
3. 运行: `python -m http.server 8000`
4. 浏览器访问: `http://localhost:8000`

### 任何平台
1. 直接双击 `index.html` 文件
2. 在浏览器中打开即可使用

## 🎨 第一次绘画

### 步骤1: 选择工具
- 点击左侧工具栏的画笔图标
- 画笔工具会高亮显示

### 步骤2: 调整设置
- 点击颜色选择器选择喜欢的颜色
- 拖动滑块调整画笔大小(1-50像素)

### 步骤3: 开始绘画
- 在中间的白色画布上按住鼠标左键
- 拖动鼠标绘制线条
- 松开鼠标完成绘制

### 步骤4: 尝试其他工具
- 橡皮擦: 擦除不需要的部分
- 直线: 绘制精确的直线
- 矩形: 绘制矩形框
- 圆形: 绘制圆形
- 编辑: 调整已绘制线条的弧度

## 🌟 核心功能体验

### 体验1: 线条弧度编辑(独特功能!)
1. 用画笔画一条曲线
2. 点击编辑工具(铅笔图标)
3. 点击刚才画的线条
4. 拖动蓝色圆点调整弧度
5. 体验精确控制线条的乐趣!

### 体验2: 图层管理
1. 点击右侧"添加图层"按钮
2. 在新图层上绘制不同内容
3. 尝试上移/下移图层
4. 点击眼睛图标隐藏/显示图层
5. 感受专业绘图软件的图层管理!

### 体验3: AI配色优化
1. 完成一幅简单的画作
2. 点击"AI优化配色"按钮
3. 等待2秒钟
4. 查看AI为你优化的配色方案
5. 如果不满意,可以点击撤销重试!

### 体验4: 自动保存
1. 绘制一些内容
2. 等待5秒钟
3. 观察底部出现的"已自动保存"提示
4. 刷新浏览器页面
5. 你的作品依然存在!

## 💾 保存你的作品

### 导出工程
1. 点击"导出工程"按钮
2. 浏览器会下载一个JSON文件
3. 保存文件到安全的位置

### 导入工程
1. 点击"导入工程"按钮
2. 选择之前导出的JSON文件
3. 你的作品会完整恢复

## 🎯 常用操作

### 撤销操作
- 点击"撤销"按钮
- 可以撤销最多50步操作

### 清除画布
- 点击"清除画布"按钮
- 确认后清除所有内容

### 清除当前图层
- 点击"清除当前图层"按钮
- 只清除当前选中的图层

## 🔧 故障排除

### 问题: 画布不显示
**解决方案**: 
- 确保使用现代浏览器(Chrome/Firefox/Safari/Edge)
- 检查JavaScript是否启用
- 尝试刷新页面

### 问题: 工具不工作
**解决方案**:
- 确保选择了正确的工具
- 检查画笔大小是否设置合适
- 尝试重新选择工具

### 问题: 自动保存不工作
**解决方案**:
- 检查浏览器是否允许本地存储
- 尝试清除浏览器缓存
- 使用隐私模式可能会禁用自动保存

### 问题: AI优化失败
**解决方案**:
- 确保网络连接正常
- 检查当前图层是否有内容
- 尝试刷新页面后重试
